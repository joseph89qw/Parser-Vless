# Parser-Vless
🚀 Асинхронный загрузчик HTTP/HTTPS-источников с автоматическим извлечением конфигураций VLESS, Trojan и Hysteria2, поддержкой Base64, JSON (Xray), YAML (Clash/Sing-box) и очисткой нерабочих источников.
## ✨ Возможности

- ⚡ Асинхронная загрузка (до 50 потоков)
- 🌐 Поддержка HTTP/HTTPS источников
- 🔍 Извлечение конфигов из:
  - 📝 обычного текста
  - 🔐 Base64 (включая вложенный)
  - 📦 JSON (Xray / outbounds)
  - 📄 YAML (Clash / Sing-box)
- 🔁 Резервный “Happ-метод” при сбоях
- 🧹 Авто-очистка нерабочих источников
- 📊 Реальный live-прогресс в консоли
- 💾 Сохранение всех найденных конфигов

---

## 📡 Поддерживаемые протоколы

- 🔵 `vless://`
- 🟣 `trojan://`
- 🟡 `hysteria2://` / `hy2://`

---

## ⚙️ Как работает

### 1️⃣ Загрузка источников
Скрипт читает файл:

sources.txt

✔ игнорирует пустые строки  
✔ игнорирует комментарии (`#`)

---

### 2️⃣ Параллельная обработка

Используется:

- `asyncio`
- `aiohttp`
- до **50 потоков**

⚡ Все источники обрабатываются одновременно

---

### 3️⃣ Получение данных

Сначала пробуется обычный запрос:

🌐 HTTP GET (aiohttp)

Если не удаётся:

🔁 включается резервный режим — **Happ Method**
- `requests`
- кастомный User-Agent
- HWID header
- повторные попытки

---

## 🧠 Извлечение конфигов

### 🔎 1. Прямой поиск

Ищутся строки:

- `vless://`
- `trojan://`
- `hysteria2://`

---

### 🔐 2. Base64 декодирование

- автоопределение base64
- декодирование текста
- повторная проверка результата
- поддержка вложенного base64

---

### 📦 3. JSON (Xray / Outbounds)

Поддерживает:

- `outbounds`
- `config.outbounds`

Конвертация в:

➡ VLESS  
➡ Trojan  
➡ Hysteria2  

---

### 📄 4. YAML (Clash / Sing-box)

Поддержка:

- `proxies:`

Типы:

- vless
- trojan
- hysteria2

## 🧹 Работа с ошибками

Если источник невалиден:

❌ добавляется в:

blacklist_sources.txt

❌ удаляется из:

sources.txt

---

## 💾 Выходные файлы

### 📄 configs.txt
Все найденные конфиги (включая дубликаты)

---

### 🚫 blacklist_sources.txt
Плохие / нерабочие источники

---

### 📡 sources.txt
Обновлённый список источников

---

## ➕ Локальные конфиги

Если есть файл:

my_configs.txt

✔ добавляется в итоговый список  
✔ проверяется на валидность  
✔ включается в `configs.txt`
